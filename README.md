# 동화 속으로 :cherry_blossom:
2019 JCC IT show 전시작  

## 01. 스토리 설명

>학교 도서관에서 우연히 책을 펼치게 된 주인공!  
동화 속으로 빨려 들어가게 되는데...  
주인공은 무사히 현실 세계로 돌아올 수 있을 것인가!
